# **App Name**: CraftMyCV

## Core Features:

- Resume Template Selection: Provide templates inspired by universities like MIT and Harvard to quickly format a resume
- Data Entry Forms: Form fields for users to input personal details, educational background, work experience, and skills.
- AI-Powered Optimization: Analyse resume content using Gemini AI to identify areas for improvement based on selected job roles and industry standards; the tool will identify potentially missing information, and offer suggestions for improving impact
- In-App Editor: Allows the user to make basic changes, so that the final generated document doesn't need post-processing with external software
- Document Generation & Download: Generate and offer resume as Word (.docx) and/or PDF for download.
- ATS Checker: Offer an ATS checker, so the user gets immediate feedback

## Style Guidelines:

- Primary color: "#606c38" to convey professionalism and trust.
- Background color: "#fefae0" to provide a clean and modern feel.
- Accent color: "#bc6c25" for call-to-action buttons and highlights, offering a subtle, sophisticated contrast.
- Body font: 'Inter', sans-serif, for clear, professional reading
- Headline font: 'Space Grotesk', sans-serif, for impactful headlines
- Use a consistent set of professional icons from a modern library (e.g., FontAwesome or Remix Icon) to visually represent sections and skills.
- Implement a clean, grid-based layout for form fields, templates, and preview sections.
